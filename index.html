Snake vs CPU

How to Play

Race against the CPU snake to eat the red food. Each food eaten adds 1 point. Be careful not to hit the walls, yourself, or the CPU snake!

Keyboard Controls:

  • Arrow Keys: Move snake
  • Enter or Escape: Pause/Resume game
  • Enter: Restart after Game Over

Gamepad Controls:

  • D-Pad/Analog Stick: Move snake
  • Start Button: Pause/Resume game
  • Start Button: Restart after Game Over
